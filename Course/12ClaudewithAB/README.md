## 与 Claude API 课(c11)的关系

**这两门课高度重合,是同一套内容的两个平台版本。** `Course/07ClaudeApi/`(**c11**,73 课)直接调 Anthropic API,本课走 AWS Bedrock。1–61 课的主题几乎一一对应。

**只学一门就够,除非你确实要在 Bedrock 上部署。** 若两门都看,重点关注本课独有的部分:

- **控制模型输出**(10)、**批量工具调用**(31)、**灵活的工具提取方式**(33)
- RAG 部分多出**重排序**(42)与**上下文检索**(43)两课
- 收尾多出 **Computer Use**(68–69)与**并行运行 Claude Code**(66)、**自动化调试**(67)

反过来,c11 独有的有: 网络搜索工具、代码执行与 Files API、四种工作流模式(并行/链式/路由)、工作流 vs 智能体。

## 课程结构

1. **认识与上手**(1–3)—— Claude 是什么、怎么开始对话、怎么把结果问得更好
2. **产品形态**(4)—— Chat / Cowork / Code 三种入口的区别
3. **核心功能**(5–10)—— Projects、Artifacts、Skills、工具连接、企业搜索、Research 模式
4. **落地与收尾**(11–14)—— 分角色的用法示例、其他使用方式、总结、测验

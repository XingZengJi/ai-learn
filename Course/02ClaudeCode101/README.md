## 课程结构

1. **认识与装机**(1–4)—— 它是什么、怎么运作、装好、发出第一条指令
2. **核心工作流**(5–7)—— **Explore → Plan → Code → Commit** 这条主线,加上上下文管理与代码审查
3. **定制体系**(8–12)—— 五种把 Claude Code 调成「你的」工具的手段: **CLAUDE.md、Subagents、Skills、MCP、Hooks**
4. **测验**(13)

**第 5 课的工作流是全课的骨架**,后面的定制手段都是在这条流水线上加装东西。

## 课程结构

这门课是 AI Fluency 系列里**最像实操工作坊**的一门:它有一个**贯穿全程的真实项目**——「社区诊所候诊时间查询器」,从第 4 课一直做到第 8 课。

1. **打底**(01–03)—— 建造者简报 → 4D 框架 → 生成式 AI 原理与局限
2. **委派**(04)—— **写零行代码的一课**: 问题简报 + 委派计划 + 验收测试
3. **描述**(05)—— **描述链**: 用户原话 → 需求 → 技术规格 → AI 指令 → 测试,然后真的构建 + 找人演示
4. **辨别**(06–07)—— 先用**五个镜头**看代码,再用**四条准则**看体验
5. **审慎**(08)—— 五问发布清单,部署 + 三个真实用户 + 一次迭代
6. **收尾**(09)—— 4D 合一,挑一件本周的真事去做

**练习是连续的,不是各自独立的**: 第 1 课的建造者简报在第 2 课被映射到 4D;诊所项目从第 4 课的三份交付物开始,每课往前推一步,到第 8 课真的发布。

## 几条值得单独记住的

**委派实现是安全的,委派判断不是**(01)—— 全课复现率最高的一句,也是切分任务的那把刀。

**大多数 AI 失败,回溯起来是更早阶段的描述、辨别或审慎失误**(01)—— 解释了为什么这门课几乎不谈提示词技巧: **输出不对时,病灶通常在你提要求之前就埋下了。**

**内环 / 外环**(02)—— 描述&辨别管日常每一次交互,委派&审慎管「要不要用、怎么用」的大决策。**比并列罗列四个 D 好记,因为它给了明确的触发时机。**

**描述链: 用户原话 → 需求 → 规格 → AI 指令 → 测试**(05)—— **提示词只是第四环。** 结果不对时回去改提示词,可能一直在错误的环节上迭代。配套的一句: **AI 听不到用户没说出口的话。**

**每一个形容词都要是一个你能辩护的决定**(05)—— 快、简单、清晰。**每个没被量化的形容词,都是你把判断权默默转交给了实现者。**

**辨别的五个镜头**(06)—— 正确性、质量、**契合度**、体验、**责任**。后两格最常被整个跳过,而它们恰好最依赖只有你才掌握的语境。

**AI 的盲点: 并发、安全、只在规模上来后才崩的东西**(06)—— 共同特征是**在开发环境里全都是隐形的**。

**「做得好看点」是愿望,不是规格**(07)—— 以及 **AI 默认不会把无障碍做对**: 因为它学的是网上代码的平均水平,而平均水平的无障碍不及格。**这条可推广到所有「行业普遍做得差」的东西。**

**你拥有的是结果,不是产出**(08)—— **「AI 写的」什么都解释不了,也开脱不了任何事。**

**上线自有一套词汇,你不问它不会提**(08)—— **数据迁移、版本管理、限流、功能开关**。可以直接把这几个词拿去问 AI,成本极低、信息量很大。

**随便做原型,谨慎地发布**(08)—— AI 把原型成本压到接近零,**于是「做出来了」和「应该上线」之间的界限模糊了**。保持产出廉价,但**把评估标准维持在原来的高度**。

**4D 是动态的,不是一个顺序**(09)—— 素养的标志不是按顺序走完四步,**而是能随时判断出「现在卡住的是哪一个 D」**。

## 与其他 AI Fluency 课程的关系

本仓库同系列课程:

- `Course/09FluforEdu/` —— 教育者版
- `Course/10FluforStu/` —— 学生版
- `Course/14T-AIFluency/` —— Teaching AI Fluency(如何把 4D 教给别人)
- `Course/15AIFAIfornon/` —— 公益组织版
- `Course/19AIF4SB/` —— 小微企业版
- `Course/20AIforB/`(本课)—— 建造者版

**与小微企业版(19AIF4SB)的对照**最能说明这门课的特点:

| | 小微企业版 | 建造者版(本课) |
|---|---|---|
| 练习形态 | 各课独立练习,围绕**你自己的生意** | **一个项目贯穿五课**(诊所候诊查询器) |
| 辨别的对象 | 法规、价格数据、截止日期(**信息准确性**) | 代码与 UX(**产物质量**),分五个镜头 |
| 组织政策 | 有(第 8 课单设一页纸 AI 政策) | 无 |
| 交付概念 | 无 | **有: 发布清单、迁移/限流/功能开关、主动下线** |
| 独有工具 | 内环/外环 | **描述链、五个镜头、三种协作模式** |
| 合作方 | PayPal 等 | **CodePath** |

第 3 课(AI 的能力与局限)是 `Course/18AICandL/` 的浓缩版,与 19AIF4SB 第 3 课内容基本一致——想深入直接看那门课。

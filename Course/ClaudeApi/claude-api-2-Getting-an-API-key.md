# Building with the Claude API - 2 Getting an API key 获取 API 密钥

> Course: Building with the Claude API · Lesson 2
> 课程: Building with the Claude API · 第 2 课
> 来源: Anthropic Academy (anthropic.skilljar.com),用户登录后提供英文原文,2026-07-17 整理翻译存档

In the next video we will be making a request to the Anthropic API. To do so, you will need a secret API key. This guide will walk you through the process of creating an API key.

> 下一节视频我们会向 Anthropic API 发起一次请求。要做到这一点,你需要一个密钥(API key)。这份指南会带你走一遍创建 API 密钥的流程。

## Step One: Navigate to the Anthropic API Console 第一步:进入 Anthropic API 控制台

In your browser, navigate to https://console.anthropic.com/ and log in to your Anthropic account. You'll then see a page like this:

> 在浏览器中打开 https://console.anthropic.com/ ,登录你的 Anthropic 账号,登录后会看到控制台首页。

## Step Two: Click the 'Get API Keys' button 第二步:点击「Get API Keys」按钮

This button can be found towards the top right of the main dashboard page.

> 这个按钮位于主控制台页面的右上角。

## Step Three: Click the 'Create Key' button 第三步:点击「Create Key」按钮

At the top right of the page, find the 'Create Key' button and click it.

> 在页面右上角找到「Create Key」按钮并点击。

## Step Four: Enter a workspace and name for your key 第四步:为密钥选择工作区并命名

Create the key in workspace 'Default' and enter a name for your key. This name is used to help you identify the keys you generate. Let's use a name of 'Anthropic Course'.

> 在「Default」工作区里创建这个密钥,并给它起一个名字,方便你日后识别自己生成过的密钥。这里我们用「Anthropic Course」作为名字。

## Step Five: Copy the Key 第五步:复制密钥

Your API key will then be displayed in a pop up window. Copy this key and hold onto it - we will use it in the next video. This key will only be displayed once, so make sure you copy it!

> 密钥会在一个弹窗中显示出来。请复制并妥善保存——下一节视频会用到它。这个密钥只会显示这一次,一定要记得复制!

If you accidentally close the window, delete the old key and generate it again.

> 如果不小心把弹窗关掉了,把这个旧密钥删掉,重新生成一个新的就行。

对产品经理来说,这个密钥就像是「一次性显示的银行卡密码」——平台出于安全考虑,不会让你事后再翻出来看,所以务必当场存好(比如放进密码管理器或 `.env` 文件),而不是想着「等下再复制」。
